## 🚀 Community Dashboard (GSoC Contribution)

A simple frontend dashboard built to interact with MetaCall FaaS APIs.

### Features:
- View deployed functions using `/api/inspect`
- Trigger deployments using `/api/deploy/create`
- Simple UI for developers

### Demo Project:
https://github.com/mraniket404/metacall-dashboard

This improves developer experience and makes MetaCall easier to use.
